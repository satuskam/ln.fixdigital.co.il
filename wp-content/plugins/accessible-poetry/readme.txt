== Description ==
WordPress Accessibility Plugin that provide solutions to common accessibility issues in websites. can help you keep your website more in line with the WCAG 2 and to expose your website to people with disabilities. 

 What does the plugin include?

 * Accessibility Toolbar with buttons to change the font size, the contrast, disable animations and more.
 * Beautiful Skiplinks that let your user navigate between main areas of the page via keyboard.
 * Find which images don't have Alternative text and alert you.
 * Outline effect for objects in focus mode.
 * Add empty ALT attribute to images with no ALT at all.
 * Add underline to links.
 * Add your own custom css or js code for accessibility.


This plugin is provided by Amit Moreno to help make the internet more accessible for people with disabilities. please aware that the plugin author is not responsible to the user or to any third party for any direct or indirect damage of any kind from any use of this plugin.

== Installation ==

1. Download the link.
2. Upload the zip file via the Plugin upload.
3. Activate the plugin.
4. Go to the "Accessiblity" admin page and follow the instructions there.
